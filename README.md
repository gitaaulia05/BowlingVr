# 🎳 Bowling VR

[![Unity Version](https://img.shields.io/badge/Unity-6.0-blue)](https://unity.com/)

**Bowling VR** adalah simulasi game bowling berbasis virtual reality yang menghadirkan pengalaman bermain realistis dan menyenangkan. Meskipun berbasis VR, game ini dapat dimainkan **tanpa headset VR**, sehingga siapa saja bisa merasakan sensasi berada di arena bowling virtual.

---

## 🎮 Cara Bermain

1. Buka project di Unity.  
2. Gunakan controller dengan klik kanan untuk menembak bola
